### Requirements and development setup

- [VirtualBox](https://www.virtualbox.org/)
- [Vagrant](http://www.vagrantup.com/)
- [Ansible](http://www.ansible.com/)


To test uppon cloning:

    vagrant up
    vagrant provision
    vagrant ssh onelove
    bin/runserver.sh
